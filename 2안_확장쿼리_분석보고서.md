# 2안 확장 쿼리 분석 보고서

- **작성일**: 2026-01-11(update)
- **목적**: 1안(원본+영문) 대비 2안(확장+영문)의 쿼리 설계 및 결과 비교 분석

---

## 1. 개요

### 1.1 배경
- 1안은 기존에 정의된 **원본 키워드**를 사용하여 NTIS 데이터를 추출
- 2안은 1안에 **관련 세부기술 키워드**를 추가하여 재현율(Recall) 향상 시도
- 두 안 모두 **동일한 노이즈 제거 조건**(AND 필터)을 적용

### 1.2 쿼리 구성 비교

| 구분 | 1안 | 2안 |
|------|-----|-----|
| 원본 키워드 | ✅ | ✅ |
| 세부기술 확장 키워드 | ❌ | ✅ |

> **핵심**: 1안과 2안의 유일한 차이는 **세부기술 확장 키워드 포함 여부**

## 5. 첨부: 최종 쿼리 코드

> **참고**: 모든 쿼리는 **한글 + 영문(Dimensions 기반)** 키워드를 포함

### 5.1 AI 반도체 (2안)

```python
# 핵심 키워드 (OR) - 한글 + 확장
ai_core_kr = df['검색텍스트'].str.contains(
    'AI반도체|인공지능반도체|AI칩|인공지능칩|AI프로세서|뉴럴프로세서|신경망처리장치|뉴로모픽|AI가속기', 
    case=False, regex=True
)

# 영문 키워드 (Dimensions 기반)
ai_core_en = df['검색텍스트'].str.contains(
    'AI semiconductor|Artificial intelligence chip|AI processor|Neural processing unit', 
    case=False, regex=True
)

# 조건부 키워드 (AND)
npu_mask = df['검색텍스트'].str.contains('NPU', case=False)
npu_cond = df['검색텍스트'].str.contains('반도체|칩|프로세서|하드웨어|가속기|신경망', case=False, regex=True)

pim_mask = df['검색텍스트'].str.contains('PIM', case=False)
pim_cond = df['검색텍스트'].str.contains('메모리|반도체|프로세서|연산|인메모리', case=False, regex=True)

# 최종 조합
ai_mask = ai_core_kr | ai_core_en | (npu_mask & npu_cond) | (pim_mask & pim_cond)
```

### 5.2 전력 반도체 (1안 권고 - 2안 확장 효과 없음)

```python
# 핵심 키워드 (OR)
power_core_kr = df['검색텍스트'].str.contains(
    '전력반도체|파워반도체|전력소자|MOSFET', 
    case=False, regex=True
)

# 영문 키워드 (Dimensions 기반)
power_core_en = df['검색텍스트'].str.contains(
    'power semiconductor|power device|power transistor|power MOSFET', 
    case=False, regex=True
)

# SiC/GaN 조건부
sic_mask = df['검색텍스트'].str.contains('SiC', case=False)
sic_cond = df['검색텍스트'].str.contains('전력|파워|반도체|MOSFET|인버터|컨버터', case=False, regex=True)
gan_exact = df['검색텍스트_raw'].str.contains(r'\bGaN\b', case=True, regex=True)
gan_kr = df['검색텍스트'].str.contains('질화갈륨', case=False)

# 최종 조합
power_mask = power_core_kr | power_core_en | (sic_mask & sic_cond) | gan_exact | gan_kr
```

### 5.3 첨단 패키징 (2안)

pkg_core_kr = df['검색텍스트'].str.contains(

    case=False, regex=True
)

# 영문 키워드 (Dimensions 기반)

)

### 5.4 망간 기반 양극재 (1안/2안 동일)

```python
# 핵심 키워드 (OR)
mn_core_kr = df['검색텍스트'].str.contains('과리튬', case=False)

# 영문 키워드 (Dimensions 기반)
mn_core_en = df['검색텍스트'].str.contains(
    'Li-rich|Mn-rich cathode|lithium manganese oxide|overlithiated', 
    case=False, regex=True
)

# AND 조건
mn_cathode = df['검색텍스트'].str.contains('망간') & df['검색텍스트'].str.contains('양극재')
mn_battery = df['검색텍스트'].str.contains('망간') & df['검색텍스트'].str.contains('양극') & \
             df['검색텍스트'].str.contains('전지|배터리|리튬', regex=True)

# 최종 조합
mn_mask = mn_core_kr | mn_core_en | mn_cathode | mn_battery
```
